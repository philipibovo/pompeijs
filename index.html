O que é pompei.js?

O pompei.js é uma biblioteca que surgiu da necessidade de ter mais agilidade ao criar layouts, baseado em colunas responsivas ganhamos em agilidade ao estruturar automaticamente o código HTML.

Por ser totalmente adaptável ao tamanho da tela, é possível infinitas combinações de colunas, tornando assim um sistema de grid totalmente personalizável e responsivo, com um aplicação extremamente simples e rápida.

É possível usar colunas redimensionadas de acordo com a largura do elemento pai, como também é possível usar colunas com largura fixa, sendo cada uma da mesma proporção em % pré estabelecida. Assim teremos layouts diferentes para cada tipo de tela (Smartphone, Tablet e Computador).

2. Inclua em seu arquivo HTML a biblioteca JQuery

HTML

<script type="text/javascript" src="pompei-0.0.1.min.js"></script>

Como funciona?

O tamanho de cada coluna vai depender da quantidade de colunas presente dentro de uma linha.

Cada linha 'pp-row' vai ter 100% de largura do elemento logo acima dele. O tamanho de cada coluna 'pp-col' é calculado em tempo de execução. Para obter a largura de cada coluna, a largura da linha é dividida pela quantidade colunas, com isso você terá colunas com a mesma largura, independente quantidade de colunas.

No exemplo abaixo temos uma linha 'pp-row' com 5 colunas 'pp-col'.

Opções

Coluna mais larga

Para que uma coluna tenha uma largura maior basta adicionar a classe 'pp-colspan-*' ao elemeto 'pp-col', onde '*' é a quantidade de partes que a coluna vai ocupar.

Por exemplo, se tem uma linha com 9 colunas, mas uma coluna tem a classe 'pp-colspan-2', essa coluna terá a largura de 16.66666667 enquanto as outras 8 colunas terão largura de 8.33333333%. Totalizando assim 100% da largura da linha

No exemplo abaixo temos uma linha 'pp-row' com 3 colunas 'pp-col' sendo que a segunda coluna tem largura de 4 colunas.

Margem

Com o pompei.js, por padrão, cada coluna tem 5 pixels de margen de lado esquerdo e 5 pixels de margem do lado direito, totalizando assim 10 pixels de margens laterais.

É possível setar 0 para as margens das colunas. Para isso use a classe 'pp-no-margin'.

A classe 'pp-no-margin' pode ser aplicada em cada coluna ou ser aplicada diretamente na linha.

Quando a classe 'pp-no-margin' é aplicada na coluna, somente está coluna ficará sem as margens laterais, as demais colunas da linha continuarão com margens laterais.

Quando a classe 'pp-no-margin' é aplicada diretamente na linha, todas as colunas desta linha ficarão sem margens laterais.

Espaço Interno

Com o pompei.js, por padrão, cada coluna tem 10 pixels de espaço interno do lado esquerdo, 10 pixels de espaço interno do lado direito, 5 pixels de espaço interno em cima e 5 pixels de espaço interno embaixo. Somando assim 20 pixels espaço interno laterais e 10 pixels de espaço interno na vertical.

É possível setar 0 para os espaços interno das colunas. Para isso use a classe 'pp-no-padding'.

A classe 'pp-no-padding' pode ser aplicada em cada coluna ou ser aplicada diretamente na linha.

Tipos de telas

Baseado na largura da tela, é possível criar diferentes layouts para cada tipo de tela, por exemplo: um layout para telas de smartphones, um para telas de tablets e para telas de computadores.

Smartphones (sm): telas de até 575px de largura.

Tablets (md): telas entre 576px até 991px de largura.

Computadores (lg): telas maiores que 991px.

Coluna mais larga por tipo de tela

É possível determinar que uma coluna seja mais larga apenas em um tipo de tela, basta inserir a classe 'pp-colspan-{TipoDeTela}-*', onde {TipoDeTela} é 'sm' para smartphone, 'md' para tablets e 'lg' para computadores.

Com os tamanhos pré-definidos podemos exibir colunas de larguras diferentes de acordo com cada tipo de telas.
